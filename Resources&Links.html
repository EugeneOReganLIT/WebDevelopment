Resources and Links

The purpose of this page is to list the various resources and links that have made many of the contributors' coding experience easier. If you would like to add or comment on any of the content, please get in touch via our Contact Us page or send an email directly to my email account.

Equipment

  • A4 sketchpads - Good for rough work and diagrams.
  • Whiteboards - While maybe seen as overkill, a whiteboard can be very helpful for group discussions and working out problems initially.
  • A4 foolscap paper - To write down and refine the rough code.
  • Multi-coloured pens - To denote different elements in the code e.g. comments, method calls, etc.

Books

It is always beneficial to have a physical book nearby when programming. As well as being easier to read than several webpages, you can mark pages for quick reference. Nowadays, most of these books are downloadable in .pdf format, making storage and reading easier.
It is important to note that while there are 9 iterations of Java, there is a strong compulstion to get the most recent texts. Books covering previous versions of Java will be sufficient for any beginner / intermediate programmer.

  • Beginning Programming with Java For Dummies, 3rd Edition,Burd, B.A. - Another in the popular series of books aimed particularly at beginners.
  • OCA Java SE 8 Programmer I Certification Guide 1st Edition, Gupta, M. - Although written for the OCA exams, Gupta covers the subjects expertly and in an accessible manner. An excellent author.
  • Java in 21 Days, Sams Teach Yourself (Covering Java 9) (8th Edition) Cadenhead, R. - A good reference text, in it's 8th guise. It is well organised, with distinct sections and clear path of progression.
